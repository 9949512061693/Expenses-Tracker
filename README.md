# 💰 Income & Expense Tracker

A clean and responsive web application that helps users track their income and expenses efficiently. This project focuses on real-time data updates, interactive UI, and meaningful financial insights using charts.

---

## 🚀 Features

- 📌 Add and manage transactions (income & expenses)
- 📊 Dynamic dashboard with:
  - Total Balance
  - Monthly Income
  - Monthly Expenses
  - Savings Rate
- 📈 Interactive **Bar Chart** (Income vs Expenses)
- 🥧 **Doughnut Chart** for category-wise expense analysis
- 💾 Data persistence using **localStorage**
- 📱 Fully responsive UI (mobile + desktop)
- 🎯 Clean and modern user interface

---

## 🛠️ Tech Stack

## ⚙️ How It Works

1. Users add a transaction with:
   - Description
   - Amount (+income / -expense)
   - Category

2. Data is stored in **localStorage**, so it persists even after refresh.

3. The app dynamically updates:
   - Income & Expense totals
   - Balance calculation
   - Savings percentage

4. Charts update automatically:
   - Bar chart → overall financial comparison
   - Doughnut chart → category-wise expense breakdown
